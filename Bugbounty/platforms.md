# Bug bounty

Locations to look for Bug bounty hunting.

## Google search

Ask your standard search engine for "[ORGANIZATION] vulnerability disclosure program" e.g. 'Tesla'.

```
[ORGANIZATION] vulnerability disclosure program
```

## Bug bounty platforms
