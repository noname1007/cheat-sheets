# HTML template

a basic html template.
```html
<!DOCTYPE html>
<!-- comment  -->
<html>
        <head>
                <meta charset="utf8">

                <title>title</title>

                <link rel="icon" type="image/x-icon" href="favicon.ico">
        </head>

        <body style="background-color:black;">

                <div id="text">
                        <center><h1 style="color: white;">TEXT</h1></center>
                </div>
                <div id="link">
                        <center><a href="http://127.0.0.1:8080/index.html">LINK</a></center>
                </div>
        </body>
</html>
```
